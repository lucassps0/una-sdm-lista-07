Exercícios: API de Calculadora de Impacto Ambiental (EcoTrack) Contexto O mundo corporativo e os governos estão focados em metas de Net Zero (emissão líquida zero de carbono). Para que as pessoas e empresas tomem decisões melhores, elas precisam de dados rápidos.
Sua missão é criar o primeiro "tijolo" de uma plataforma de monitoramento ambiental. 
O Desafio Você deve criar uma nova API em .NET 10 chamada EcoTrackApi.
O objetivo é criar um endpoint que forneça uma "Dica de Sustentabilidade" aleatória ou um cálculo simples de emissão baseada em km rodados. Requisitos Técnicos 1. Criação do Controller: Crie um arquivo chamado SustentabilidadeController.cs. Centro Universitário 2. Rota: A rota para acessar deve ser [controller]. 3. Método GET: O método deve retornar uma string contendo uma recomendação ambiental. 4. Lógica Opcional (Diferencial): Tente fazer com que o método retorne uma frase diferente de uma lista (Array) toda vez que for chamado. 
Realize o teste da sua API usando o Bruno, ou Postman ou Swagger ou via navegador. Tire um print dela funcionando.
